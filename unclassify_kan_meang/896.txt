@si ppt ยินดีมากครับ ขอบคุณที่พูดกันอย่างสุภาพด้วยเหตุผลนะครับ ^^ ผมเองเคยไปนั่งฟังคุณอรรถวิชช์ที่ม๊อบไล่ตระกูลชิน ก็ยอมรับว่าค่อนข้างชื่นชอบในบุคลิกและเหตุผลของท่านครับ 

แต่ในคลิปนี้ผมค่อนข้างไม่เห็นด้วยกับเขานะครับ   เอามาที่ประเด็นคดีหมิ่น สิ่งที่ผมไม่เคยเหนเลย คือการพิสูจน์ว่าสิ่งที่เขาพูด หรือคลิปที่มันเปนอะไรที่หมิ่นๆกัน มันจริงหรือไม่ เราทำเหมือนทุกอย่างเป็นข้อมูลเท็จ 100% และยืนบน Mindset นี้เป็นหลัก โดยไม่เคยคิดเลยว่า มันอาจจะจริงก็ได้ 
ซึ่งถามว่ามันสำคัญตรงไหน สำหรับผม ถ้า Description ในการมีอยู่ของสถาบัน คือการเป็นที่เคารพรักและนับถือ การเป็นที่น่านับถือมันก็สำคัญนะครับ และการน่านับถือมันน่าจะมาจากใจ และการจะเกิดสิ่งนั้น การไม่กังขา น่าจะเป็นสิ่งสำคัญครับ ^^

การที่คุณเทิดทูนสถาบัน หรือเดือดเนื้อร้อนใจ ผมก็ไม่ได้มีปัญหาอะไรครับ ผมก็ยอมรับว่าน่าจะมีอยู่ แต่สำหรับผม การหมิ่นประมาท มันเป็นการกระทำระหว่างบุคคลต่อบุคคล ไม่มีผลทางกายภาพอะไรต่อระบบ เด็กมันด่าสถาบันขนาดไหน การบริหาร ระบบอะไรต่างๆมันก็ Run ต่อไป ไม่ได้หยุดติดขัดอะไร คดีก็ว่ากันต่อไป ทุกอย่างยัง Run ตามเรื่องได้ดี ดังนั้นผมว่า โทษของการพูดหมิ่น ที่ไม่กระทบต่อระบบการ Run สิ่งต่างๆของประเทศเลย ไม่น่าจะใช่อาญาความมั่นคงนะครับ ^^ การหมิ่นมีมานานแล้ว อย่างที่พวกคุณพูด แต่ประเทศมันก็ไปของมันตามปกติ การที่ระบบต่างๆอย่างเช่นเศรษฐกิจมันเกิดติดขัด หรือเกิดมีภัยธรรมชาติ หรือการที่รถไฟฟ้ามันจะเสีย หรือเพื่อนบ้านมารุกราน  ผมว่าไม่น่าจะเกี่ยวกับการหมิ่นนะครับ   

คดีแพ่งมีโทษของมันอยู่ และการปรับก็ยืดหยุ่นตามแต่ละบุคคลอยู่แล้ว คุณด่าผม ถ้าคุณผิดจริงคุณอาจจะจ่ายเงินสักหมื่นนึง แต่ถ้าคุณด่าณเดชน์ ญาญ่า คุณไม่น่าจะจ่ายเรตนี้แน่ๆครับ แถมแฟนคลับก็ยังฟ้องแทนกันไม่ได้ด้วยนะครับ ^^ และแถมยังมีโทษจำคุกอยู่ด้วย เมื่อคดีสิ้นสุด คนผิดรับโทษ เจ้าทุกข์ได้ชื่อเสียงคืน มันน่าจะจบตามนั้น ผมว่าการหมิ่นมันน่าจะเข้าตามทรงนี้มากกว่าครับ  

การรักสถาบันไม่ใช่เรื่องเสียหายครับ แต่ถ้าโทษมันต้องเพิ่มจนเกินจริง เพื่อให้พวกคุณหายเจ็บใจหรือสะใจ หรือถ้าใครไม่เป็นอันทำมาหากิน เพราะรู้สึกว่าคนหมิ่นมันรับโทษไม่พอ  ผมว่าอันนี้น่าจะต้องจัดการกับความรู้สึกตัวเองนะครับ และถ้ามีคนที่ทนไม่ไหว เพราะโทษมันไม่หนักพอ ไม่สาแก่ใจ แล้วมาตั้งขบวนการมาล้มประเทศ หรือต้องฆ่าอีกฝั่งเพราะแค้นจนทนไม่ไหว ผมว่าไอ้ขบวนการนี้ต่างหาก ที่มันเป็นปัญหาครับ 

มองกันอีกมุม ถ้าคุณมองว่ากษัตริย์เป็นสมมติเทพ ในสมัยที่โลกเรา สถาบันกษัตริย์รุ่งเรื่องที่สุด คนที่จะจัดการราชการแทนกษัตริย์ได้ ต้องได้รับการแต่งตั้ง จะส่งหนังสือราชการฟ้องเอาโทษได้ ต้องเป็นผู้เดินสารที่ได้รับการแต่งตั้ง ใครบอกว่าเป็นผู้เดินสารโดยที่ไม่ได้รับการแต่งตั้ง น่าจะหัวกุดนะครับ เผลอหัวกุดทั้งตระกูลด้วย แต่ตอนนี้ สถาบันไม่ได้เดือดร้อนอะไร แต่ดูเหมือนว่าทุกคนเป็นผู้เดินสารได้ทั้งหมด ทุกอย่างมันเล่นใหญ่เกินไปไหมครับ 

สิ่งที่มันควรจะเป็น ส่วนตัวผมว่าการจะกำจัดการหมิ่นได้อย่างถาวรที่สุด ไม่ใช่การห้ามพูดและปราศจากการพิสูจน์ แต่เป็นการเอามาพิสูจน์ไปเลย ว่าคลิปโน้นๆนี้ๆ สิ่งที่คนหัวหงอกหรือคนนั้นๆนี้ๆพูด มันเป็นเท็จ ถ้ามันพิสูจน์ได้อย่างเป็นธรรม ผลออกมาคนเห็นและรู้แจ้ง ถึงตอนนั้น 112 มันไม่จำเป็นต้องมีด้วยซ้ำครับ
